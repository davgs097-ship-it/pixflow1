FAQ

Perguntas frequentes

Preciso saber programar para usar o PayOS?▶
Não. O PayOS foi feito para infoprodutores e vendedores digitais sem conhecimento técnico. O Integrador de Sites faz tudo automaticamente — você só cola o HTML.
Quais gateways PIX são suportados?▶
Atualmente suportamos Zero One Pay, Paradise Pags e GhostsPays. Você pode usar os 3 simultaneamente no plano Pro, com produtos diferentes em cada gateway.
O PayOS cobra taxa por transação?▶
Não. O PayOS cobra apenas uma mensalidade fixa. As taxas de transação são cobradas diretamente pelo gateway (Z1, Paradise ou GhostsPays), não por nós.
Como funciona o upsell automático?▶
Você cadastra uma URL de redirecionamento no produto. Assim que o pagamento PIX for confirmado pelo gateway, o cliente é redirecionado automaticamente para a URL de upsell — sem intervenção manual.
Posso usar no plano gratuito sem limite de tempo?▶
Sim. O plano Starter é gratuito para sempre, com 1 gateway e até 3 produtos. Você só precisa fazer upgrade se quiser múltiplos gateways, produtos ilimitados e o integrador de sites completo.
